# MATLAB Rules for Cursor IDE

This directory contains Cursor rules for MATLAB development.

## Installation

Copy the rules file to your project's `.cursor/rules/` directory:

```bash
mkdir -p .cursor/rules
cp cursor/rules/matlab-development.mdc .cursor/rules/
```

Or add to your global Cursor rules directory.

## What's Included

- `matlab-development.mdc`: MATLAB coding rules covering style, testing, optimization, error handling, and documentation standards.

## Note

For full interactive slash commands, use the Claude Code plugin. These rules provide persistent context to Cursor's AI for MATLAB development.
